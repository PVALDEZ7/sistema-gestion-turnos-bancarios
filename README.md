# Sistema de Gestión de Turnos para Atención Bancaria

Este proyecto consiste en una aplicación web para organizar y controlar los turnos de atención dentro de una entidad bancaria.

El sistema busca reducir el desorden y mejorar el tiempo de atención de los clientes.

- Pedro Efraín Valdez Galloso
- David Gabriel Caimatta Soria

## Funciones principales

- Inicio de sesión
- Generación automática de turnos
- Selección del tipo de atención
- Visualización de la cola de espera
- Llamado del siguiente turno
- Finalización de la atención
- Consulta del historial
- Generación de reportes

## Tipos de atención

- Consultas
- Depósitos
- Retiros
- Pago de servicios
- Atención al cliente
